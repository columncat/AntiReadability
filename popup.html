AntiReadability

v1.0.0
이 페이지에서는 확장 프로그램을 실행할 수 없습니다 (크롬 보안 페이지 등).

사용 방법

  1. 1 웹페이지에서 마우스 우클릭 후 '민감 정보 난독화 시작'을 선택하거나 위의 '시작' 버튼을 클릭합니다.
  2. 2 화면에 나타난 플로팅 툴바에서 원하는 난독화 유형을 고릅니다.
  3. 3 숨기고 싶은 텍스트 영역을 마우스로 드래그하면 즉시 난독화됩니다. (이미지는 '이미지' 모드 선택 후 클릭)
  4. 4 난독화를 원래대로 되돌리려면 '지우개' 모드를 선택한 후, 난독화된 텍스트를 드래그하거나 이미지를 클릭하여 복원합니다.
  5. 5 작업 완료 후 '완료'를 눌러 툴바를 닫고 화면 스크린샷을 찍습니다. (단축키 Ctrl+Z / Ctrl+Y 지원)
